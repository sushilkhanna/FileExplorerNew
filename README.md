# File Explorer

Web-based file management system built using Spring Boot.

## Features
- Browse folders
- Upload files
- Download files
- File sorting
- Directory navigation

## Tech Stack
Java
Spring Boot
REST APIs
HTML
CSS

## API Endpoints
GET /files
POST /upload
GET /download/{file}

## Project Architecture
User Interface
↓
Spring Boot Controllers
↓
Service Layer
↓
File System Operations
